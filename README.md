-	Industrial Automation, IoT, IIot, Industry 4.0, Big Data, Artificial Intelligence, Machine Learning, Deep Learning, Data Science and Analytics.
-	Enterprise Integration
-	GIS and Location Aware Systems.
-	Agile and DevOps
